# Error Intelligence Layer — Documentation

## Overview

`error-intelligence-layer` is a production-grade npm package for Node.js (TypeScript-first) that transforms **any** thrown value into a structured, enriched, developer-friendly object.

The library answers four questions about every error:

| Question                 | Answered by                          |
| ------------------------ | ------------------------------------ |
| **What happened?**       | Normalization + parsing              |
| **Where did it happen?** | Stack parsing                        |
| **Why did it happen?**   | Root cause extraction + intelligence |
| **How to fix it?**       | Suggestion engine                    |
